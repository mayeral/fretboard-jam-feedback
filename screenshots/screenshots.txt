SECTION: App Modes — Basics
----------------------------------------------------------------------

mode-notehunt.png
  What to show : Note Hunt mid-game. A note name (e.g. "D") in the
                 prompt card. Fretboard interactive, no fret highlighted
                 yet (before the player taps).
  Tips         : String lock OFF, full neck visible.

mode-fretflash.png
  What to show : Hear & Name mode. The fret position shown in the prompt
                 card (e.g. "3rd fret, A string"). The 12 note-name choice
                 buttons visible below the fretboard.
  Tips         : Capture before an answer is selected so all buttons
                 show their neutral state.

mode-octaveconnect.png
  What to show : Octave Connect mode. A note name shown (e.g. "E").
                 The 6 string-progress dots visible at the top of the
                 prompt card (some done in green, one active in purple).
  Tips         : Complete 2–3 strings first so the progress indicator
                 is clearly visible.

feature-string-lock.png
  What to show : Note Hunt with the string lock active on the A string
                 (string 5). Only the A string is highlighted / active

feature-fret-range.png
  What to show : The settings toolbar open, Min Fret set to 5 and
                 Max Fret set to 9. The fretboard should show the
                 fret range highlighted (frets 5–9 lit up in the
                 range-highlight color).
  Tips         : Capture the toolbar and fretboard together in one shot.

feature-mic-mode.png
  What to show : Mic mode active (microphone icon lit/highlighted in
                 the prompt card toolbar). A note prompt visible.
                 Ideally the mic meter or pitch indicator visible.
  Tips         : Grant mic permission in the browser first.
                 The 🎤 icon should appear in its active/on state.

SECTION: Theory

theory-fretboard-layout.png
  What to show : The bare fretboard (no game active or between rounds)
                 with inlay dots clearly visible at frets 3, 5, 7, 9
                 and the double dot at fret 12.
  Tips         : Dark mode works great here — the inlay dots stand out
                 against the dark neck. No highlighted notes, just the
                 clean neck with markers.

theory-shortcuts.png
  What to show : A note highlighted on the low E string (e.g. fret 3 = G)
                 AND its octave counterpart on the D string (fret 5 = G)
                 both lit up simultaneously, illustrating the
                 "2 strings down, 2 frets up" shortcut pattern.
  Tips         : Use the Octave Connect mode — play the first string,
                 then take the screenshot before moving to the next string
                 so both positions are highlighted at once.
                 Annotate with an arrow in an image editor if possible.
